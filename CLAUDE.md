## Sincronización (multi-dispositivo)

**Al empezar sesión:** ejecuta siempre `git pull` antes de hacer cualquier cosa.
**Al terminar sesión:** ejecuta `git add -A && git commit -m "session: <resumen de 1 línea>" && git push` para no perder progreso entre ordenadores.

## Estado actual

Lee `STATUS.md` al empezar para retomar el contexto — mantenerlo actualizado con el estado y los próximos pasos acordados.
